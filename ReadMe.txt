
-------------------------- APP ------------------------------

Step 1

Go to following url
https://github.com/roshanjayawardena/temperature-converter

Step 2

Run following command using cmd or gitbash and clone the project.
git clone --branch master https://github.com/roshanjayawardena/temperature-converter.git

Step 3

Then open the APP project by using visual code and 
run npm install

Step 4

run ng serve
